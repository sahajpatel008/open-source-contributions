# Contributions

This folder contains detailed writeups for individual open source contributions.

Each file should focus on one meaningful contribution or closely related set of changes. Use [`_template.md`](_template.md) when adding a new writeup.

## Naming Convention

Use lowercase, descriptive file names:

```text
project-name-short-contribution-title.md
```

Examples:

```text
react-fix-accessibility-warning.md
nextjs-docs-routing-example.md
nodejs-add-regression-test.md
```
